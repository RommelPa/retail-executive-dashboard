# Retail Executive Dashboard: Business Performance Monitoring with Streamlit

## Overview

This project builds an interactive executive dashboard to monitor retail sales performance.

The goal is to transform transactional sales data into a decision-oriented dashboard with KPIs, trends, product analysis, country analysis, customer concentration, and executive insights.

## Business Context

Executives need fast visibility into revenue performance, sales concentration, customer behavior, and operational risks.

A dashboard should not only display charts. It should help decision-makers understand what is happening, where performance is concentrated, and what requires attention.

## Objectives

- Load and clean retail transaction data.
- Build a reproducible data preparation pipeline.
- Create executive KPIs.
- Visualize revenue trends.
- Analyze revenue by country.
- Analyze top products.
- Measure customer concentration.
- Build an interactive Streamlit dashboard.
- Document how to run and maintain the project.
